## 📖 설명

다양한 console 사용법

## 📌 배운 점

- 새로운 발견: `assert`, `count`

## 🧩 주요 코드

```js
// Testing
const p = document.querySelector("p");
console.assert(p.textContent.includes("oops"), "That is wrong!");

// counting
console.count("Q");
console.count("Q");
console.count("A");
console.count("A");
```
